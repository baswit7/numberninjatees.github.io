Privacy Policy

This Privacy Policy applies to the NumberNinjaTees website and the NumberNinjaTees Marketing OS use case for authorized marketing of the NumberNinjaTees brand.

Effective: 24 May 2026Owner: Bas WitIJmuiden, The Netherlands

01Who We Are

NumberNinjaTees is an independent apparel brand owned by Bas Wit in IJmuiden, The Netherlands. We create data and coding themed apparel sold through our official Etsy sales channel at numberninjatees.etsy.com.

This website is a business information and integration disclosure site. It is not a social media platform and it does not provide public user accounts.

02Website Hosting

This public website is hosted through GitHub Pages. When a visitor loads these static pages, GitHub may process technical request information according to GitHub's own applicable terms and privacy practices.

NumberNinjaTees does not operate a website visitor database through this static site.

03Etsy Purchases

Products are offered through Etsy. Shopping, payment, orders, messages and fulfilment interactions conducted on Etsy are governed by Etsy's policies and the applicable Etsy shop terms, not by data collection on this static website.

04Pinterest API And OAuth

NumberNinjaTees Marketing OS is intended only for marketing the brand's own products. Where Pinterest API access is authorized, Pinterest OAuth tokens are used only to access and manage authorized NumberNinjaTees marketing resources, such as the brand's own boards, pins and performance information.

  • We do not access another Pinterest account without that account owner's explicit OAuth authorization.
  • We do not request or collect Pinterest passwords or session cookies.
  • We do not scrape Pinterest and we do not sell Pinterest data or personal information.
  • Tokens and secrets are not shown on this public website and are not committed to this public GitHub Pages repository.
If an authorized operator chooses to enter API credentials in a bring-your-own-key Marketing OS interface, those credentials are stored locally in that operator's browser using localStorage. This public informational website does not contain a credentials input form and does not receive those locally stored values.

05Use And Retention

Authorized Pinterest data is used only to create, manage or evaluate NumberNinjaTees marketing content. Any credentialed integration must retain only what is operationally required and must allow access to be disconnected or revoked through the relevant integration/Pinterest account controls.

06No Sale Of Data

NumberNinjaTees does not sell personal data or Pinterest data. We do not make Pinterest-derived account information available to third parties for their marketing purposes.

07Contact

Questions about this policy, NumberNinjaTees or the Pinterest API use case can be sent to contact@numberninjatees.com.

Business identity: NumberNinjaTees, owner Bas Wit, IJmuiden, The Netherlands. See the dedicated Pinterest API use case page and contact page for additional review information.
